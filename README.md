# VibeCoder 🛑✨

**The Anti-Prettier for true trench coders.**

Let's be real. You just copy-pasted a function from an AI agent, and now your codebase is infected with `🚀`, `💡`, and comments explaining what a `console.log` does. It's embarrassing. You're losing your aura. 

**VibeCoder** is a VS Code extension that intercepts your `Ctrl + S` and aggressively de-GPTs your code before it hits the disk. It strips the cringe, kills the AI formatting, and restores that raw, "I've been debugging in the dark for 14 hours" human aesthetic.

## 🔥 Features

* **The Emoji Nuke (Live):** Instantly vaporizes 99% of standard emojis (`🚀`, `✨`, `🐛`). No more generative AI stench in your strings.
* **The Smart Scrubber (Cooking):** Parses your code into an AST to safely rip out useless AI over-explanations without breaking your URLs or logic.
* **The Chaos Engine (Cooking):** Randomly swaps single/double quotes and introduces controlled formatting anarchy (because humans aren't perfectly indented).

## 🛠️ Usage

1. Write code (or let's be honest, paste it).
2. Hit `Ctrl + S`.
3. Watch VibeCoder instantly scrub the AI vibes off your file before it even saves.

## 💻 Installation

Currently in the kitchen. 
1. Clone the repo.
2. Run `npm install`.
3. Hit `F5` in VS Code to spin up the extension testing host.
4. Try to save a file with an emoji in it. I dare you.
